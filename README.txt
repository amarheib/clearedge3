# clearedge-frontend (מוכן ל-Vercel)
1) העלה את כל התיקייה הזו ל-GitHub כריפו בשם `clearedge-frontend`.
2) ערוך את `index.html` ושנה את `window.__CLEAREDGE_API__` לכתובת ה-API שלך (מ-Render).
3) ב-Vercel: New Project → בחר את הריפו → Framework: Vite → Deploy.
4) אחרי הדיפלוי, פתח את האתר → לחץ "נסה דוגמה".
